C. Pola III

Time limit		1 s
Memory limit	64 MB

Deskripsi
Pak Dengklek meminta Anda untuk memperhatikan contoh masukan dan keluaran yang
diberikan, menemukan polanya, lalu membuat program yang menghasilkan pola
tersebut. Bisakah Anda?

Format Masukan
Baris pertama berisi sebuah bilangan bulat N.

Format Keluaran
Pola berukuran N.

Contoh Masukan 1
5

Contoh Keluaran 1
0
12
345
6789
01234

Contoh Masukan 2
7

Contoh Keluaran 2
0
12
345

1234567

Batasan
1 ≤ N ≤ 100
