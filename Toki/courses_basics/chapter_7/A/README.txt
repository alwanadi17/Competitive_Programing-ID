A. Break Continue Exit

Time limit		1 s
Memory limit	64 MB

Deskripsi
Pak Dengklek memberikan Anda sebuah bilangan bulat N. Anda diminta untuk
mencetak bilangan-bilangan bulat dari 1 sampai dengan N secara berurutan, satu
per baris, dengan aturan sebagai berikut:

Lompati bilangan kelipatan 10.
Jika program akan mencetak bilangan 42, jangan cetak 42, tetapi cetak ERROR dan
jangan cetak apa-apa lagi.

Format Masukan
Sebuah baris berisi sebuah bilangan bulat N.
