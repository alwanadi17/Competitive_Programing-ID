E. If Then / Case

Time limit		1 s
Memory limit	64 MB

Deskripsi
Pak Dengklek meminta Anda membuat sebuah program sebagai berikut. Program akan
menerima sebuah bilangan bulat N. Jika N merupakan satuan, cetak satuan. Jika N
merupakan puluhan, cetak puluhan. Jika N merupakan ratusan, cetak ratusan. Jika
N merupakan ribuan, cetak ribuan. Jika N merupakan puluh ribuan, cetak
puluhribuan.

Format Masukan
Sebuah berisi sebuah bilangan bulat N.

Format Keluaran
Sebuah baris berisi keluaran sesuai permintaan soal.

Contoh Masukan 1
4

Contoh Keluaran 1
satuan

Contoh Masukan 2
12345

Contoh Keluaran 2
puluhribuan
